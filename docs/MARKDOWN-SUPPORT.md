# 法術與物品描述欄位：Markdown / HTML 支援

> 法術詳細資訊與物品詳細訊息的**描述欄位**支援 Markdown 與有限 HTML，可直接在編輯時輸入語法，詳情頁會渲染為對應樣式。  
> 若需了解整體功能或資料結構，請見 [README.md](../README.md) 與 [code-architecture.md](code-architecture.md)。

---

## 已支援功能

- **Markdown**：粗體、斜體、列表等基本語法。
- **HTML**：例如 `<span style="...">` 用於自訂顏色與字重，適合標示傷害類型、關鍵數字等。

---

## 使用範例

### 基本 Markdown

**粗體**

```markdown
**這是粗體**
```

*斜體*

```markdown
*這是斜體*
```

列表

```markdown
- 項目 1
- 項目 2
- 項目 3
```

### HTML 顏色（自訂樣式）

紅色傷害文字（例如火焰傷害）：

```html
<span style="color: #dc2626; font-weight: bold;">12d8 火焰傷害</span>
```

藍色冷凍文字：

```html
<span style="color: #3b82f6; font-weight: bold;">冷凍傷害</span>
```

### 常用顏色參考

| 用途示例 | 色碼 |
|----------|------|
| 紅色（火焰/傷害） | `#dc2626` |
| 藍色（冷凍/寒冰） | `#3b82f6` |
| 紫色（閃電/魔法） | `#a855f7` |
| 綠色（毒素/治療） | `#10b981` |
| 黃色（光明/神聖） | `#eab308` |
| 黑色（死靈/黯蝕） | `#1e293b` |
| 白色（治療/神聖） | `#f1f5f9` |

### 混合範例

```markdown
目標受到 <span style="color: #dc2626; font-weight: bold;">12d8 火焰傷害</span>，豁免成功則減半。

**更高等級施法：**
- 使用 4 級法術位時，傷害增加 1d8
- 使用 5 級法術位時，傷害增加 2d8

*專注時間最長 1 分鐘*
```

## 實際使用方式

在法術或物品的**描述欄位**中直接輸入上述語法，儲存後在詳細資訊頁面會自動渲染。若發現某些標籤或樣式未如預期，可能為前端安全或實作限制，可查閱元件內實際使用的 Markdown/HTML 渲染邏輯。

### 建議測試步驟

1. 編輯一個法術或物品的描述。
2. 加入顏色與粗體等標記。
3. 儲存後開啟詳細資訊。
4. 確認樣式正確顯示。

---

## 注意事項

- HTML 標籤較繁瑣，適合少量、重點使用。
- 若某些模式很常用，可考慮未來擴充簡化語法或預設樣式。
- 建議以基本 Markdown 為主，需要強調顏色或字重時再使用 HTML。

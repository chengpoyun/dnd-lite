# Markdown 語法支援說明

## ✅ 已實作功能

法術詳細資訊和物品詳細訊息的描述欄位現在支援 Markdown 和 HTML 語法。

## 📝 使用範例

### 基本 Markdown 語法

**粗體文字**
```markdown
**這是粗體**
```

*斜體文字*
```markdown
*這是斜體*
```

列表
```markdown
- 項目 1
- 項目 2
- 項目 3
```

### HTML 顏色語法（自定義顏色）

紅色傷害文字
```html
<span style="color: #dc2626; font-weight: bold;">12d8 火焰傷害</span>
```

藍色冷凍文字
```html
<span style="color: #3b82f6; font-weight: bold;">冷凍傷害</span>
```

### 常用顏色參考

- 🔴 紅色（火焰/傷害）：`#dc2626`
- 🔵 藍色（冷凍/寒冰）：`#3b82f6`
- 💜 紫色（閃電/魔法）：`#a855f7`
- 🟢 綠色（毒素/治療）：`#10b981`
- 🟡 黃色（光明/神聖）：`#eab308`
- ⚫ 黑色（死靈/黯蝕）：`#1e293b`
- ⚪ 白色（治療/神聖）：`#f1f5f9`

### 混合使用範例

```markdown
目標受到 <span style="color: #dc2626; font-weight: bold;">12d8 火焰傷害</span>，豁免成功則減半。

**更高等級施法：**
- 使用 4 級法術位時，傷害增加 1d8
- 使用 5 級法術位時，傷害增加 2d8

*專注時間最長 1 分鐘*
```

## 🎯 實際應用

在法術或物品的描述欄位中，直接輸入上述語法即可自動渲染為對應樣式。

### 測試建議

1. 編輯一個法術的描述
2. 加入顏色和粗體標記
3. 儲存後查看詳細資訊
4. 確認樣式正確顯示

## ⚠️ 注意事項

- HTML 標籤較繁瑣，適合少量使用
- 如果發現某些模式很常用，未來可以開發簡化語法
- 建議先用基本 Markdown，需要特殊顏色時才用 HTML
